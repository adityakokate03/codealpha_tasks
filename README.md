# Task 1 - Simple E-Commerce Store (CodeAlpha Internship)

This project is a basic full-stack e-commerce web application built using:

Frontend:
- HTML
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- MongoDB

---

## Features

- Product Listing
- Add Products to Cart
- Shopping Cart
- Order Processing
- User Registration
- User Login
- Backend API using Express
- MongoDB Database Integration
